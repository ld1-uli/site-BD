Atelier Git & Github

DESCRIPTION

L'atelier Git & GitHub est une session pratique destinée aux étudiants d'IIBS afin de leur apprendre à utiliser
Git et GitHub pour gérer leurs projets informatiques. Les participants découvriront les bases
du contrôle de version, apprendront à créer un dépôt, effectuer des commits, travailler
avec les branches et publier leurs projets sur GitHub. Cet atelier leur permettra
d'acquérir des compétences essentielles pour le développement logiciel et le travail collaboratif.

ORGANISATION

8h00 OUVERTURE Accueil des participants, installation de Git sur les postes et vérification des comptes GitHub.
10h00 INTRODUCTION Bases du contrôle de version avec Git : création de dépôts, premiers commits et bonnes pratiques.
11h00 COLLABORATION SUR GITHUB Gestion des branches, pull requests et résolution de conflits sur un mini-projet en équipe.
14h00 FIN DE L'ATELIER Synthèse des bonnes pratiques utilisées en entreprise et session questions/réponses.
